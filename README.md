# Home Retail ETL

Учебный проект: генерация CSV-выгрузок по магазинам и кассам,
загрузка данных в PostgreSQL и автоматизация через планировщик заданий / cron.

## Структура проекта

- `src/` — Python-скрипты (генератор и загрузчик)
- `data/` — входящие выгрузки CSV
- `data/processed/` — уже загруженные файлы
- `sql/` — DDL-скрипты для БД
- `img/` — скриншоты cron/Task Scheduler и таблиц БД
- `requirements.txt` — зависимости проекта

Инструкция по развёртыванию и запуску будет дописана позже.
